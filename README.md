# 🚀 Martin James | Full-Stack Portfolio

A high-performance, responsive personal portfolio built to showcase my journey as a Full-Stack Developer. This project features a custom-built certificate slider, dynamic project grid, and a dark-themed UI optimized for all devices.

## ✨ Key Features

* **Responsive Certificate Slider:** A custom-built, mobile-friendly carousel displaying 9+ professional credentials with touch-swipe support and hover-zoom effects.
* **Modern UI/UX:** Dark-mode aesthetic featuring high-contrast "Cyber-Mint" accents and smooth-scroll navigation.
* **Fully Responsive:** Optimized for everything from ultra-wide monitors to small mobile screens using CSS Flexbox and Grid.
* **Contact Integration:** Real-time communication setup using EmailJS/Contact Form logic.
* **Firebase Deployment:** Hosted on Google Firebase for low-latency global delivery.
